# Agent Fabric protocol

This package is the sole public owner of Agent Fabric operations, schemas and
MCP descriptors. Start with these entry modules:

- [`schema.ts`](src/schema.ts) defines the protocol schemas.
- [`operations.ts`](src/operations.ts) defines public operations.
- [`mcp-projection.ts`](src/mcp-projection.ts) projects operations to MCP descriptors.
- [`operation-codecs.ts`](src/operation-codecs.ts) is the codec barrel; shared codec primitives are in [`codec.ts`](src/codec.ts).
- [`schemas/protocol.schema.json`](schemas/protocol.schema.json) and the other schema artifacts are generated by [`scripts/write-schema.mjs`](scripts/write-schema.mjs).

See [messaging and public protocol](../../docs/specs/agent-fabric/messaging-and-public-protocol.md).

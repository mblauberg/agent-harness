# Agent Fabric Console

The Console starts at [`bin.ts`](src/bin.ts) and [`cli.ts`](src/cli.ts). Its
operator UI follows the presenter-model-controller seam:
[`presenter.ts`](src/presenter.ts), [`model.ts`](src/model.ts), and
[`controller.ts`](src/controller.ts).

See [Console scope and projections](../../docs/specs/console/scope-and-projections.md).

## Domain

Agent harness comprehensive review implementation

## Mission

Implement the entire agent-harness comprehensive review cleanly on one comprehensive-review branch and one final PR: every substantiated finding, accepted decision, Lane A-D obligation, roadmap step, deterministic gate, live local verification, and required independent review; continue until the human sets STOP.

## Locked constraints (design around — do not relitigate)

D-021 chair charter; KICKOFF ordering; one branch and one PR; no merge or origin/main push; worktrees only under owning-repo .worktrees; preserve dirty root; no release deploy publish production credential registry mutation standing egress or external-effect profile; workspace-write-offline stays inert until the exact Step-3 gate; never access list or enumerate .agent-run/AFAB-004; durable truth lives in repo docs and exact per-lane receipts.

## Hard gates (no auto-accept; judge-panel + cross-family before final/promoted)

Normative authority and schema one-way doors; security and containment controls; provider-action and lifecycle custody; write-profile containment verdict; irreversible migration or compatibility decisions; Spec-05 80x24 usability adjudication; programme acceptance. Each needs objective mutation-sensitive evidence plus fresh native review and Claude Opus cross-family review; bonus-family attempt where the charter requires it.

## Escalation gates (human / expert / panel / spike / apply / promotion)

human: any genuine external-effect authority expansion or need to access AFAB-004, plus final PR review and merge; expert: none presently named; judge-panel: disputed one-way doors and final programme acceptance; spike: fixed Step-3 containment matrix and other questions with no deterministic oracle; apply: release deploy credential registry and provider-login effects remain forbidden without new human authority; promotion: only one green comprehensive-review PR may be presented and the human alone merges it.

## Build ceiling (autonomous up to here; beyond is owed/escalated)

Authorised local source docs tests fixtures schemas configs worktrees commits and deterministic mocked or local-only probes; push only comprehensive-review and open one draft/final-review PR. No merge release deploy publish provider-login change production credential or registry mutation external messaging standing network-egress enablement or other shipping action.

maintainability 0.07; human-attention efficiency 0.05.

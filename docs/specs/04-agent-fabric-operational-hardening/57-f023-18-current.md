
Baseline construction stages every normalised row and binding before atomic
publication, then runs foreign-key/integrity checks, identity/count
reconciliation, canonical path/digest queries and registry-trigger probes.
Fault injection at every staging, schema-build, binding and catalogue-record
boundary proves that incompatible prior state remains unchanged or the
complete current schema is published. `artifact-registry.v1` and
`artifact-content-read.v1` advertise only after validation passes. Spec 05
owns all Console paging, disclosure, acceptance and viewport behaviour; this
spec owns only the daemon/client capability boundary.
